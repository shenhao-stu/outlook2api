Import Accounts

Text Import

Enter accounts, one per line in email:password format.

File Upload

Upload a .txt file with one email:password per line.

CI Import

Use this endpoint in your GitHub Actions workflow to auto-import accounts.

POST /admin/api/accounts/bulk
Content-Type: application/json
Authorization: Bearer ADMIN_PASSWORD

{
  "accounts": ["user@outlook.com:pass"],
  "source": "ci"
}

curl example:

curl -X POST https://your-domain/admin/api/accounts/bulk \
  -H "Authorization: Bearer ADMIN_PASSWORD" \
  -H "Content-Type: application/json" \
  -d '{"accounts": ["user@outlook.com:pass"], "source": "ci"}'

API Documentation

Mail API

GET/domains
List supported email domains
curl https://your-domain/domains
POST/accounts
Register account (validates via IMAP)
curl -X POST https://your-domain/accounts \
  -H "Content-Type: application/json" \
  -d '{"address": "user@outlook.com", "password": "pass"}'
POST/token
Get JWT token
curl -X POST https://your-domain/token \
  -H "Content-Type: application/json" \
  -d '{"address": "user@outlook.com", "password": "pass"}'
GET/me
Get current user info (requires Bearer token)
curl https://your-domain/me \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
GET/messages
List messages (requires Bearer token, ?page=1)
curl "https://your-domain/messages?page=1" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
GET/messages/{id}
Get a specific message (requires Bearer token)
curl https://your-domain/messages/123 \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
GET/messages/{id}/code
Extract verification code from message (requires Bearer token)
curl https://your-domain/messages/123/code \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
DELETE/accounts/me
Delete current account (requires Bearer token)
curl -X DELETE https://your-domain/accounts/me \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"

Admin API

All endpoints require Bearer token or admin_token cookie.

POST/admin/api/login
Authenticate and receive admin token
curl -X POST https://your-domain/admin/api/login \
  -H "Content-Type: application/json" \
  -d '{"password": "your_password"}'
GET/admin/api/stats
Get account statistics
curl https://your-domain/admin/api/stats \
  -H "Authorization: Bearer ADMIN_TOKEN"
GET/admin/api/accounts
List accounts with pagination and filters (?page=&search=&active=)
curl "https://your-domain/admin/api/accounts?page=1&search=user&active=true" \
  -H "Authorization: Bearer ADMIN_TOKEN"
POST/admin/api/accounts
Add a single account
curl -X POST https://your-domain/admin/api/accounts \
  -H "Authorization: Bearer ADMIN_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"email": "user@outlook.com", "password": "pass"}'
POST/admin/api/accounts/bulk
Bulk import accounts
curl -X POST https://your-domain/admin/api/accounts/bulk \
  -H "Authorization: Bearer ADMIN_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"accounts": ["user@outlook.com:pass"], "source": "manual"}'
POST/admin/api/accounts/upload
Upload accounts file (multipart form)
curl -X POST https://your-domain/admin/api/accounts/upload \
  -H "Authorization: Bearer ADMIN_TOKEN" \
  -F "file=@accounts.txt"
PATCH/admin/api/accounts/{id}
Update account (toggle active status)
curl -X PATCH https://your-domain/admin/api/accounts/123 \
  -H "Authorization: Bearer ADMIN_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"is_active": true}'
DELETE/admin/api/accounts/{id}
Delete a specific account
curl -X DELETE https://your-domain/admin/api/accounts/123 \
  -H "Authorization: Bearer ADMIN_TOKEN"
DELETE/admin/api/accounts
Delete all accounts
curl -X DELETE https://your-domain/admin/api/accounts \
  -H "Authorization: Bearer ADMIN_TOKEN"
GET/admin/api/accounts/{id}/password
Retrieve account password
curl https://your-domain/admin/api/accounts/123/password \
  -H "Authorization: Bearer ADMIN_TOKEN"
GET/admin/api/export
Export all accounts as text file
curl https://your-domain/admin/api/export \
  -H "Authorization: Bearer ADMIN_TOKEN" -o accounts.txt
